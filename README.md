# Youtube_music_rank
這專案你可以選擇要找曲風、音樂人或國家來找音樂排行榜前10

程式說明：這個程式的功能主要為「排行youtube個別類型的前十名」，一開始執行的時候你有四個選項：1.各個國家"當下"的歌曲熱門排行、2.各個曲風的總觀看排行、3.各個音樂人音樂的總觀看排行、4.結束程式。
在選擇1.的時候：你可以輸入國家的代號，他會給你當下那個國家最熱門的十個歌曲排行。
在選擇2.的時候：你可以輸入你想了解的歌曲風格，他會給你那個曲風關鍵字的十個歌曲排行。
在選擇3.的時候：你可以輸入你喜歡的歌手，他會給你那個歌手youtube頻道裡面觀看數最高的十個音樂影片。
在選擇4.的時候：程式會直接結束(選擇1、2、3都會一直執行程式直到你輸入4.)
